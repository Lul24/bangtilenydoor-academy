# Bangtilenydoor Academy 🎓

An online academy platform for courses and learning management.

## 🌐 Pages

* Home – `index.html`
* Programs – `programs.html`
* Scholarships – `scholarships.html`
* Impacts – `impacts.html`
* Inquiry – `inquiry.html`
* Contact – `contact.html`

## 🚀 Features

* Course and program information
* Scholarship details
* Inquiry and contact forms
* Simple multi-page website structure

## 🛠️ Tech Stack

* HTML
* CSS (if included)
* Netlify (for deployment)

## 📦 Deployment

This project is configured for deployment using Netlify via `netlify.toml`.
